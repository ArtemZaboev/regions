# regions
Directory of regions

Приложение реализующее справочник регионов и предаставляющее REST-API на чтение и изменеие спрвочника.

Используемые технлогии:

Spring Boot;

ORM: MyBatis; 

Сборщик: Maven;

Spring Cache;

Логирование: Slf4j;

База данных: H2;

Приложение создано в качестве тестового задания.
