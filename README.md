# regions
Directory of regions

Приложение реализующее справочник регионов и предоставляющее REST-API на чтение и изменение справочника.

Используемые технологии:

Spring Boot;

ORM: MyBatis; 

Сборщик: Maven;

Spring Cache;

Логирование: Slf4j;

База данных: H2;

Приложение создано в качестве тестового задания.
